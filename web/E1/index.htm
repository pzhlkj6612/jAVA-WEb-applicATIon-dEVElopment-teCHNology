实验一 内置对象及JDBC应用程序设计

- 首先要注意所用的编码都是UTF-8；

- 然后你需要运行下列SQL语句来创建数据库：

create database university;
use university;
Create table student(
    XueHao char(7),
    Name char(8),
    Sex char(2),
    Age int,
    Jiguan char(20),
    Dept char(10)
);

- 进入学籍管理系统；
